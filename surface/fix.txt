Instructions

    (Prep) Install Dependencies:

 sudo apt install git curl wget sed

    Clone the linux-surface repo:

 git clone https://github.com/jakeday/linux-surface.git ~/linux-surface

    Change directory to linux-surface repo:

 cd ~/linux-surface

    Run setup script:

 sudo sh setup.sh
